查看Windows Update更新紀錄以及關閉更新

2025年09月30日 593 字 大概 2 分鐘

每次都被微軟的更新搞到，今天紀錄一下檢查更新紀錄方法，以及關閉更新的方式。

如何查看 Windows Update 的更新時間與成功與否

方法一：使用事件檢視器 (Event Viewer)

  1. 打開「事件檢視器」（在搜尋列輸入 事件檢視器 或 Event Viewer）。
  2. 展開左側選單到：

事件檢視器 > 應用程式與服務記錄 > Microsoft > Windows > WindowsUpdateClient > Operational

  1. 常見的事件 ID：
事件 ID 意義
19 更新成功安裝
20 更新安裝失敗
21 開始下載
22 開始安裝
31 安裝完成（需重新啟動）
  1. 點選任一事件可查看詳細資訊，包括 時間、更新名稱、結果碼 等。

方法二：Reliability Monitor（穩定性監視器）

  1. 在搜尋欄輸入 Reliability Monitor 或「穩定性歷程記錄檢視器」。
  2. 查看時間軸上的「Windows Update」事件。
  3. 點擊任一事件，可查看詳細更新記錄與錯誤訊息。

這是最直覺的 GUI 工具。

如何關閉或延後 Windows Update

方法一：從設定中暫停更新（適用於家庭版/專業版）

開啟「設定」 >「Windows Update」

點選「暫停更新」，可選擇 1～5 週不更新

方法二：透過群組原則停用更新（僅限 Windows 專業版/企業版）

執行 gpedit.msc

進入電腦設定 > 系統管理範本 > Windows 元件 > Windows Update > 管理的更新安裝
雙擊「設定自動更新」，選擇「已停用」或改為「通知下載與安裝」。

方法三：停用 Windows Update 服務（適用於所有版本，但不推薦）

執行 services.msc

找到 Windows Update 服務

右鍵 > 停用，啟動類型改為「停用」

注意：某些版本的 Windows 會在重開機後自動重啟該服務，建議與群組原則搭配使用。
