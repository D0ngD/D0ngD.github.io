oneMP3：合併音檔並自動加入封面

2025年09月24日 523 字 大概 2 分鐘

其實是很久之前寫的小工具了，在dlsite買的廣播劇不是都會依照章節分成不同音檔嗎？因為覺得這樣太多檔案所以寫了一個合併工具。

合成後的音檔會跟資料夾名稱同名，就不用再自己取名(如果你的資料夾都是DLSITE預設下載的英文+數字開頭那推薦你使用dlsite爬蟲自動改名字,網上有其他大神寫好的爬蟲可以用)

就算你的系統沒有預先安裝 ffmpeg，也能在程式裡 手動選 ffmpeg / ffprobe 路徑使用。

下載

  • 前往 GitHub Releases 下載 EXE

Windows 可能跳出 SmartScreen，點「更多資訊」→「仍要執行」。

使用方式

  1. 執行 oneMP3.exe
  2. 依序：
    • 選擇 音檔所在資料夾（內含 mp3/wav）
    • 選擇 輸出資料夾
    • 選擇 封面圖片（jpg/png）
  3. 程式會輸出 (FULL)資料夾名稱.mp3，封面已寫入為 Front Cover（ID3v2.3）。

第一次使用：設定 ffmpeg

若系統找不到 ffmpeg，程式會跳窗請你：

  • 選 ffmpeg.exe（在 FFmpeg 的 bin 目錄）
  • 程式會自動推測同路徑的 ffprobe.exe；若不存在會再請你選

建議把 FFmpeg 解壓到 C:\ffmpeg\bin\，路徑簡單不會有權限/中文路徑問題。

功能

  • 簡單合併：支援 .mp3 與 .wav，依檔名排序合併
  • 標籤相容：封面存為 ID3v2.3
  • ffmpeg 選擇器：沒裝也能用，直接指定 ffmpeg.exe 即可

常見問題

  • 封面沒顯示？
    先用 VLC 或 Mp3tag 確認封面是否嵌入成功；有時檔案總管需要重新整理快取。
  • 選到 ffmpeg 還是錯？
    確認你選的是 bin\ffmpeg.exe，且同資料夾有 ffprobe.exe；必要時移到 C:\ffmpeg\bin\ 再選。
